# Arch + Hyprland Setup

Este repositorio documenta la configuración de mi entorno en Arch Linux con Hyprland y Zsh.

## 🧠 Stack
- Arch Linux
- Hyprland (Wayland compositor)
- Kitty terminal
- Zsh + Oh My Zsh

## ⚙️ Objetivo
Crear un entorno minimalista, rápido y personalizable para desarrollo y uso diario.

## 📸 Estado actual
- Hyprland configurado con shortcuts personalizados
- Zsh con Oh My Zsh funcionando
- Keybinds básicos para navegación y gestión de ventanas
